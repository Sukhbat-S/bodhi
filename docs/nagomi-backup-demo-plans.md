# Nagomi уулзалтын нөөц төлөвлөгөөнүүд

Маргаашийн Nagomi уулзалтад WiFi тасарсан тохиолдолд сандрахгүй байхын тулд энэ гурван төлөвлөгөөг урьдаас бэлдсэн. Тус бүрт хэрэгтэй зүйл, бэлтгэл, болон 30 секундэд дараагийн төлөвлөгөөнд шилжих алхмуудыг тусгасан.

---

## Plan A — Бүрэн шууд демо

*WiFi ажиллаж байгаа тохиолдолд ашиглах үндсэн төлөвлөгөө.*

**Шаардлагатай тоног төхөөрөмж**
- Зөөврийн компьютер (бүрэн цэнэглэсэн)
- Цахилгааны кабель
- HDMI/USB-C адаптер
- Утас (нөөц hotspot болгон бэлэн байлгах)
- Ус

**Бэлтгэл**
- Демо хийх хуудсуудыг Chrome-д урьдчилан нээж, tab-уудыг зохион байгуул
- Браузерийн кэшийг цэвэрлэж, гадны мэдэгдэл, popup-уудыг унтраа
- Утасны hotspot-ыг нэр, нууц үгтэй нь мэдэж байгаарай — ямар нэгэн асуудал гарвал шууд идэвхжүүл

**Хэрэв амжилтгүй болвол юу хэлэх**
- *"Уучлаарай, холболт тогтворгүй байна — нэг мөчид асуудлыг шийдэж дараагийн хэлбэрт шилжье."*
- *"Санаа зовох хэрэггүй, надад бэлэн төлөвлөгөө бий."*
- *"Та нарт ямар ч алдаа алдагдахгүй — бүгдийг харуулна."*

**Эелдэг сэргээх алхмууд**
1. Утасны hotspot-ыг нэн даруй идэвхжүүл
2. Компьютерийг тэр hotspot-д холбо (15 секунд)
3. Хуудсыг дахин ачааллуул — ажиллавал үргэлжлүүл
4. Ажиллахгүй бол Plan B руу шилж — *"Тэгвэл бичлэгийг үзүүлье"* гэж хэл

---

## Plan B — Утаснаас бичлэг үзүүлэх

*WiFi болон hotspot хоёул ажиллахгүй болсон тохиолдолд.*

**Шаардлагатай тоног төхөөрөмж**
- Утас (бүрэн цэнэглэсэн, нисгэгч горимоос гарсан)
- Screen recording-уудыг offline-д татан авсан байх
- Lightning/USB-C кабель (нэмэлт цэнэгт)

**Бэлтгэл**
- Демо-г бүрэн дамжуулж screen recording хий — дуу + тайлбараар
- Бичлэгийг Photos болон Files апп хоёуланд хадгал
- Утасны гэрэлтэлтийг дээд хэмжээнд тавь, автомат дэлгэц унтраалгыг идэвхгүй болго

**Хэрэв амжилтгүй болвол юу хэлэх**
- *"Интернэт холбогдохгүй байна, гэхдээ бичлэгээс бүхнийг үзүүлэх боломжтой."*
- *"Энэ нь бодит хэрэглээтэй яг адил — бүх функцийг харах боломжтой."*
- *"Асуулт байвал хариулахад бэлэн байна."*

**Эелдэг сэргээх алхмууд**
1. Утсаа гаргаж бичлэгийг нэн даруй нээ
2. *"Утаснаас үзүүлье"* гэж тайвнаар хэл
3. Утсаа хүмүүст харагдахуйц байрлалд тав эсвэл дэлгэцийг эргүүл
4. Бичлэг тасарвал эсвэл дуусвал Plan C руу шилж

---

## Plan C — Хэвлэсэн дэлгэцийн зураг + амаар тайлбар

*Технологи бүхэлдээ ажиллахгүй тохиолдолд — хамгийн найдвартай нөөц.*

**Шаардлагатай тоног төхөөрөмж**
- Хэвлэсэн дэлгэцийн зурагнууд (A4, өнгөт, хавтас доторх)
- Нэрийн хуудас
- Тэмдэглэлийн дэвтэр + үзэг

**Бэлтгэл**
- Програмын гол дэлгэцүүдийг screenshot хий: нүүр хуудас, гол функц, үр дүн
- Логик дарааллаар нь дугаарлаж, хавтсанд хийж бэлэн тавь
- Тус бүрийн ард 1-2 өгүүлбэр тайлбар бич — ярих үеэ сануулагч болгон

**Хэрэв амжилтгүй болвол юу хэлэх**
- *"Технологи заримдаа гэнэтийн зочин авчирдаг — гэхдээ хамгийн чухал зүйл бол санаа."*
- *"Хэвлэмэл материал дээрээ тулгуурлан тайлбарлая — хамгийн сайн харагдах хэсгүүдийг сонгосон."*
- *"Та нар сонирхолтой гэж үзвэл дараа нь бүрэн демо хийж өгнө."*

**Эелдэг сэргээх алхмууд**
1. Хавтсаа нэн даруй гаргаж, хуудсуудыг тарааж тав
2. *"Хэвлэсэн хувилбараасаа явья"* гэж тайвнаар хэл
3. 1-р хуудаснаас эхлэн хялбар, итгэлтэй тайлбарла
4. Хуудас бүрийн дараа асуулт байгаа эсэхийг асуу — яриаг хоёр талын болго

- [ ] Хэвлэсэн дэлгэцийн зурагнууд дугаарлагдаж хавтсанд
- [ ] HDMI / USB-C адаптер цүнхэнд
- [ ] Цахилгааны кабель цүнхэнд
- [ ] Нөөц hotspot SIM идэвхтэй (эсвэл утасны hotspot тохиргоо хийгдсэн)
- [ ] Нэрийн хуудас
- [ ] Тэмдэглэлийн дэвтэр + үзэг
- [ ] Ус
- [ ] Chrome tab-ууд нээгдсэн, кэш цэвэрлэгдсэн
- [ ] Автомат дэлгэц унтраалга идэвхгүй (компьютер болон утас хоёуланд)

> **Санаж байгаарай:** Аль төлөвлөгөөнд ч итгэлтэй, тайван байх нь хамгийн чухал. Бэлтгэл сайн хийгдсэн — чи бэлэн.
